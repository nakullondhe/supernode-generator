
#  supernode-generator

### Description
Simple Nodejs Template generator for removing repetitive creation of Nodejs projects.

## Usage

### With NPM

    npm i supernodeman
    supernodeman
    
### 	With Yarn
    yarn add global supernodeman
    supernodeman
    
### 	Templates available
 - Express-Mongoose
 - Express Mongoose Production Beta WIP (Competely usable,exludes custom error and tests)
